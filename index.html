Arbitos Redesign

Arbitos is a mobile app that allows you to purchase diesel from the comfort of your own home. You can also buy investment plans from the app and earn high returns.

Role

UI/UX Designer

Duration

3 weeks

Tools

Figma, Adobe Illustrator

Project Goal

The goal is to inprove site infrastructure to simplify user flows and make key functions more accessible. Adding significant new feature, with the goal of increasing user retention and engagement.

The Challenge

Over first challenge is to improve the user experience of the mobile app and to provide a better user experience through visual design.

The Solution

Poorly designed UI and high inflation rate is a major challenge which most user face as the money save are constantly devalued

My role

I worked on the project with four creative designers from beginning to end, and we cooperatively worked on these project phases;

  • User research
  • User experience design
  • User interface design
  • Prototyping
  • Testing

The Approach

This project covered a 12-week period. A meeting with the project manager and stakeholders to discuss the project served as the project's official launch. We, as the design team, started working after we were given the project documentation. We used a heuristic evaluation method, which is a quick and effective way to solve problems or make decisions, keeping in mind the time frame provided by the stakeholders. We categorized our thought process into:

  • Understanding the users’ needs (empathize)
  • Competitive analysis research.
  • Wireframing
  • Testing/Iterating
  • Providing a final solution
  • Prototyping
